로그인 (JWT 토큰 발급)

> Request

URL
POST /api/authenticate
Host: http://3.38.104.80:8080
Authorization: Bearer {ACCESS_TOKEN}
Parameter
Name Type Description Required
username String 앱 로그인시 사용할 기사번호 O
password String 앱 로그인시 사용할 비밀번호 O

> Response

Name Type Description Required
token String 서버와 교신시 필수로 필요한 JWT토큰 O

사용자 정보 (로그인 필수)

현재 로그인된 사용자의 정보를 불러온다.

> Request

URL
GET /api/users
Host: http://3.38.104.80:8080
Authorization: Bearer {ACCESS_TOKEN}

> Response

Name Type Description Required
id String DB 관리를 위한 기사 고유 id O
username String 앱 로그인시 사용할 기사번호 O
nickname String 기사 이름 O
authorityDtoSet Set 사용자가 가지고있는 권한 목록 O
authorityName Set 사용자가 가지고있는 권한 이름 O

검색 사용자 정보 (로그인 필수)(ROLE_ADMIN 보유)

username(기사번호)로 사용자를 검색해 해당 사용자의 정보를 불러온다. [ROLE_ADMIN 권한을 가진 사용자만 접근 가능]

> Request

URL
GET /api/users/{username}
Host: http://3.38.104.80:8080
Authorization: Bearer {ACCESS_TOKEN}
Parameter
Name Type Description Required
userId Integer 검색할 사용자의 고유 id O

> Response

Name Type Description Required
id Integer DB 관리를 위한 기사 고유 id O
username String 앱 로그인시 사용할 기사번호 O
nickname String 기사 이름 O
authorityDtoSet Set 사용자가 가지고있는 권한 목록 O
authorityName Set 사용자가 가지고있는 권한 이름 O

번호판 인증 (로그인 필수)

Number 번호판을 가진 차량이 DB에 등록되어있는지 확인하고 True 또는 False를 반환한다.

> Request

URL
GET /api/cars/numbers/{number}
Host: http://3.38.104.80:8080
Authorization: Bearer {ACCESS_TOKEN}
Parameter
Name Type Description Required
number String 등록 되어있는지 확인할 번호판 O

> Response

Name Type Description Required
{} Boolean True or False O

주입 로그 추가 (로그인 필수)

현재 로그인된 사용자의 주입 로그를 추가한다.

> Request

URL
POST /api/fill-logs/users
Host: http://3.38.104.80:8080
Authorization: Bearer {ACCESS_TOKEN}
Parameter
Name Type Description Required
pumpId String 사용한 주입기 기기의 고유 id(시리얼넘버) O
amount Double 요소수 등 제품(외 기타) 주입량 O
carNumber String 요소수 등 제품을 주입한 차량의 차량번호(번호판) O

> Response

Name Type Description Required
datetime Datetime 로그 생성 시점의 날짜, 시간(주입일시) O
pumpId Integer 사용한 주입기의 고유 id O
amount Double 주입한 액체의 주입량 O
carNumber String 차량번호 O
username String 주입기를 이용한 기사의 기사번호 O
branchName String 이용한 주입기의 회사명 O
branchCeo String 이용한 주입기 회사의 대표명 O
branchAddress String 이용한 주입기 회사의 주소 O
branchTEL String 이용한 주입기 회사의 연락처 O
approvalNumber Integer 승인번호 (미구현) O
pumpNumber Integer 이용한 주입기 번호 O
product String 주입한 제품 O
